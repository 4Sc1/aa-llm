# AA-LLM Repository

Welcome to the AA-LLM repository. This repository is a comprehensive resource for our study:

Differential Reactivity of Affect and Self-Esteem in Borderline Personality Disorder to Daily Events: Contextual Insights from Large Language Models in Ambulatory Assessment

## Repository Contents

1. **Source:** The source code created for our study.

2. **Results:** In-depth results of our study.

## Usage Guidelines

The folders contain`README.md` files with specific instructions and information about their contents. Please refer to these individual ReadMes for more detailed information.

Additionally, `LICENSE.txt` files are present in the folders. We strongly urge you to read these license files before utilising, modifying, or distributing the respective contents.
